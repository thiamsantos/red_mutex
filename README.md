## Installation

The package can be installed
by adding `red_mutex` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:red_mutex, "~> 0.1.0"}
  ]
end
```

The docs can be found at [https://hexdocs.pm/red_mutex](https://hexdocs.pm/red_mutex).
